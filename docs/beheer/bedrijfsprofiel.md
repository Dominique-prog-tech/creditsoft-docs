# Bedrijfsprofiel

Op het bedrijfsprofielscherm beheert u de basisgegevens van uw eigen kantoor: naam, adres, BTW, logo, contactinformatie.

## Het scherm openen

Klik in de sidebar onder **Beheer** op **Bedrijfsprofiel**.
