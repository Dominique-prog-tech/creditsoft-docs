# Dashboard

Het dashboard is uw startscherm na het aanmelden. Het geeft een snel overzicht van uw lopende dossiers, taken en actuele werkpunten.

## Het dashboard openen

Klik in de sidebar bovenaan op **Dashboard**. Dit is ook het scherm dat standaard verschijnt na het aanmelden.
