# Notarissen

Notarissen verlijden de notariële akte bij hypothecaire kredieten. CreditSoft houdt uw netwerk van notarissen overzichtelijk bij.

## Het scherm openen

Klik in de sidebar onder **CRM** op **Notarissen**.
