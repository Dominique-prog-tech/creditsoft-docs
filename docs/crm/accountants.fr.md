# Comptables

Les comptables sont des partenaires importants pour le courtier en crédit. Ils fournissent souvent les documents financiers nécessaires à la demande de crédit.

## Ouvrir l'écran

Cliquez dans la sidebar sous **CRM** sur **Comptables**.
