# CRM — Vue d'ensemble

Le module CRM regroupe tous les contacts avec lesquels vous collaborez en tant que courtier en crédit : les clients (emprunteurs), mais aussi les partenaires professionnels tels que les comptables, les agences immobilières, les notaires et les estimateurs.

## Contenu du module CRM

| Composant | Description |
|---|---|
| [Entreprises](companies.md) | Entreprises clientes et entreprises partenaires |
| [Contacts](contacts.md) | Personnes individuelles — emprunteurs et personnes de contact |
| [Comptables](accountants.md) | Comptables et cabinets comptables |
| [Agences immobilières](real-estate-agents.md) | Agents immobiliers |
| [Notaires](notaries.md) | Études notariales |
| [Estimateurs](appraisers.md) | Estimateurs immobiliers |
| [Biens immobiliers](properties.md) | Biens immobiliers pouvant servir de garantie |
| [Rapports CRM](reports.md) | Rapports sur votre base de données de contacts |

## Ouvrir la section CRM

Dans la sidebar, cliquez sur **CRM** pour développer la section, puis sur le composant souhaité.

!!! tip
    De nombreux enregistrements CRM sont créés automatiquement depuis un dossier de crédit :
    lors de la création d'un nouveau dossier, vous pouvez immédiatement créer un nouvel emprunteur
    ou un bien immobilier. La création séparée via ces écrans CRM est surtout utile
    pour gérer les données de base de manière proactive.
