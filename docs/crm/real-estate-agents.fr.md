# Agences immobilières

Les agences immobilières sont des partenaires essentiels pour les crédits hypothécaires. Elles vous apportent des biens immobiliers et des acheteurs.

## Ouvrir l'écran

Cliquez dans la sidebar sous **CRM** sur **Agences immobilières**.
