# Estimateurs

Un estimateur immobilier détermine la valeur marchande d'un bien immobilier. Ce rapport d'estimation est obligatoire pour les crédits hypothécaires.

## Ouvrir l'écran

Cliquez dans la sidebar sous **CRM** sur **Estimateurs**.
