# Notaires

Les notaires signent l'acte notarié pour les crédits hypothécaires. CreditSoft vous permet de gérer votre réseau de notaires de manière structurée.

## Ouvrir l'écran

Cliquez dans la sidebar sous **CRM** sur **Notaires**.
