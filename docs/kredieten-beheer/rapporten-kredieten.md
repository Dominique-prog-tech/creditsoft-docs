# Rapporten kredieten

CreditSoft biedt een breed gamma kredietgerelateerde rapporten voor interne opvolging, externe rapportage en compliance.

## Het scherm openen

Klik in de sidebar onder **Kredieten** op **Rapporten kredieten**.

    Typische rapporten:

    - **Productierapport** — overzicht van afgesloten dossiers per periode
    - **Pipeline-rapport** — lopende dossiers per fase
    - **Commissierapport** — uitbetaalde en te ontvangen commissies
    - **Tussenpersonen-rapport** — performance per tussenpersoon
    - **FSMA-rapport** — wettelijk verplichte rapportage
    - **NBB-rapport** — Nationale Bank rapportage indien van toepassing

## Een rapport genereren

1. Selecteer het type rapport.
2. Stel de parameters in (periode, filters).
3. Klik op **Voorbeeld** om te bekijken, of **Afdrukken** / **Exporteren**.
