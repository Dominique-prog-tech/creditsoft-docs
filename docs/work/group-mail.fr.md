# E-mail groupé

L'e-mail groupé vous permet d'envoyer un e-mail à un groupe de destinataires en une seule opération — par exemple à tous vos intermédiaires ou clients d'un groupe donné.

## Ouvrir l'écran

Cliquez dans la sidebar sous **Travail** sur **E-mail groupé**.

## Composer un e-mail groupé

!!! info "TODO"
    Étapes :

    1. Sélectionnez le groupe de destinataires (intermédiaires, clients, filtre personnalisé).
    2. Choisissez un [modèle d'e-mail](../settings/mail-templates.md) ou composez vous-même.
    3. Personnalisez avec des champs de fusion (nom, info dossier).
    4. Ajoutez des pièces jointes.
    5. Envoyez, ou planifiez pour un envoi ultérieur.
