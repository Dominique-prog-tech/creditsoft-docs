# Profil d'entreprise

L'écran du profil d'entreprise vous permet de gérer les données de base de votre propre cabinet : nom, adresse, TVA, logo, coordonnées.

## Ouvrir l'écran

Cliquez dans la sidebar sous **Configuration** sur **Profil d'entreprise**.
