# Comptes e-mail

CreditSoft peut envoyer et recevoir des e-mails. Cet écran vous permet de gérer les comptes e-mail utilisés par la plateforme.

## Ouvrir l'écran

Cliquez dans la sidebar sous **Configuration** sur **Comptes e-mail**.
