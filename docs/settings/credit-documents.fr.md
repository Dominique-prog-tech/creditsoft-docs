# Documents de crédit (configuration)

Ici, vous gérez les **types** de documents de crédit utilisés dans les dossiers : leur dénomination, leur caractère obligatoire et les validations applicables.

## Ouvrir l'écran

Cliquez dans la sidebar sous **Configuration** sur **Documents de crédit**.

## Différence avec les documents dans un dossier

!!! note
    Sur **cet écran**, vous gérez les **types** de documents (les modèles).
    Les documents eux-mêmes sont ajoutés **dans un dossier** —
    voir [Gérer les documents](../credit-files/documents.md).
