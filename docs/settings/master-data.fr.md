# Données de base

Les données de base sont les listes de choix et catégories utilisées partout dans CreditSoft : types de crédits, types de biens immobiliers, types de documents, professions, langues.

## Ouvrir l'écran

Cliquez dans la sidebar sous **Configuration** sur **Données de base**.

## Ajouter une valeur

1. Sélectionnez la catégorie.
2. Cliquez sur **Nouveau**.
3. Complétez les données (code, description, langues éventuelles).
4. Cliquez sur **Enregistrer**.
