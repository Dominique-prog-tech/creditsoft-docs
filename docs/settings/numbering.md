# Nummering

CreditSoft genereert automatisch unieke nummers voor dossiers, facturen, offertes en andere objecten. Op dit scherm stelt u de nummeringsformaten in.

## Het scherm openen

Klik in de sidebar onder **Beheer** op **Nummering**.
