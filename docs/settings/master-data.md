# Stamgegevens

Stamgegevens zijn de keuzelijsten en categorieën die overal in CreditSoft gebruikt worden: types kredieten, types panden, document-types, beroepen, talen.

## Het scherm openen

Klik in de sidebar onder **Beheer** op **Stamgegevens**.

## Een waarde toevoegen

1. Selecteer de categorie.
2. Klik op **Nieuw**.
3. Vul de gegevens in (code, omschrijving, eventuele talen).
4. Klik **Opslaan**.
