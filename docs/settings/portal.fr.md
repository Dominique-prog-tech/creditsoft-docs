# Paramètres du portail

Le portail de CreditSoft fournit des informations aux intermédiaires et aux clients via un environnement en ligne sécurisé. Cet écran vous permet de gérer les paramètres du portail.

!!! info "Module en développement"
    La fonctionnalité du portail est en cours d'élaboration. Cette page sera
    complètement décrite dès que le module portail sera finalisé.

## Ouvrir l'écran

Cliquez dans la sidebar sous **Configuration** sur **Portail**.
