# Congés et jours fériés

Cet écran vous permet de gérer les congés de vos collaborateurs et les jours fériés légaux que CreditSoft utilise dans les calculs d'agenda, de délais et de durées.

## Ouvrir l'écran

Cliquez dans la sidebar sous **Configuration** sur **Congés et jours fériés**.

## Effet sur l'agenda et les délais

!!! tip
    Les jours de congé et les jours fériés sont exclus des calculs en jours ouvrables,
    p. ex. pour les délais de traitement ou les échéances automatiques.
