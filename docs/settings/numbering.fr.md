# Numérotation

CreditSoft génère automatiquement des numéros uniques pour les dossiers, factures, devis et autres objets. Cet écran vous permet de configurer les formats de numérotation.

## Ouvrir l'écran

Cliquez dans la sidebar sous **Configuration** sur **Numérotation**.
