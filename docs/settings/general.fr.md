# Paramètres généraux

Les paramètres généraux contiennent les configurations à l'échelle de la plateforme : paramètres régionaux, devise, valeurs par défaut, intégrations.

## Ouvrir l'écran

Cliquez dans la sidebar sous **Configuration** sur **Paramètres généraux**.

## Catégories de paramètres

!!! info "TODO"

    Catégories typiques :

    - **Régional** : langue, pays, fuseau horaire, devise
    - **Dates et nombres** : paramètres de format
    - **Documents par défaut** : types par défaut, langues
    - **Sauvegarde** : planning de sauvegarde, emplacement
    - **Sécurité** : exigences de mot de passe, délais d'expiration de session
    - **Intégrations** : clés API pour les services externes

## Qui peut modifier ces paramètres

!!! warning
    Les paramètres généraux sont accessibles aux utilisateurs disposant de **droits d'administrateur**. Les utilisateurs ordinaires ne voient pas cet écran ou seulement en mode lecture.
