# Créer un ticket de support

Lorsque vous rencontrez un problème ou avez une question, vous pouvez créer un ticket de support directement depuis CreditSoft.

## Créer un ticket de support

1. Cliquez en bas de la sidebar sur **Créer un ticket de support**.
2. Complétez le formulaire :
    - **Sujet** : brève description du problème ou de la question.
    - **Catégorie** : type de demande (bug, question, suggestion, formation).
    - **Priorité** : faible, normale, élevée, urgente.
    - **Description** : explication détaillée, étapes pour reproduire le problème.
    - **Pièces jointes** : captures d'écran, messages d'erreur, données d'exemple.
3. Cliquez sur **Envoyer**.

## Conseils pour un bon ticket de support

!!! tip
    - **Soyez précis** : "Message d'erreur lors de l'enregistrement du dossier 12345" vaut mieux que "Ça ne fonctionne pas".
    - **Mentionnez les étapes** pour reproduire le problème.
    - **Joignez des captures d'écran** des messages d'erreur.
    - **Indiquez ce que vous attendiez** comme résultat.
