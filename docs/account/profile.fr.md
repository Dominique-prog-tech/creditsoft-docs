# Mon utilisateur

Sur votre page de profil, vous gérez vos paramètres personnels, vos préférences et votre mot de passe.

## Ouvrir l'écran

Cliquez en bas de la sidebar sur **Mon utilisateur**.

## Modifier le mot de passe

1. Cliquez sur **Modifier le mot de passe**.
2. Saisissez votre mot de passe actuel.
3. Saisissez deux fois le nouveau mot de passe.
4. Cliquez sur **Confirmer**.

!!! tip
    Choisissez un mot de passe d'au moins 10 caractères avec un mélange de lettres, chiffres et symboles.
    N'utilisez pas de mots de passe que vous utilisez également ailleurs.
