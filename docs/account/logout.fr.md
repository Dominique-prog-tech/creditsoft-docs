# Fermer la session

Lorsque vous n'utilisez plus CreditSoft activement, fermez votre session. Cela protège vos données, en particulier sur les postes de travail partagés.

## Fermer la session

Cliquez en bas de la sidebar sur **Fermer la session**. CreditSoft ferme votre session et retourne à l'écran de connexion.

## Bonnes pratiques

!!! tip
    - Fermez toujours votre session avant de laisser votre ordinateur sans surveillance.
    - Évitez de partager vos identifiants de connexion. Demandez un compte personnel à votre administrateur.
    - Fermez également les autres onglets de navigateur ou applications contenant des données sensibles.
