# Vue d'ensemble LOA

L'aperçu LOA affiche tous les dossiers avec une Letter of Acceptance — l'engagement formel d'une institution financière d'accorder le crédit.

!!! info "Qu'est-ce que le LOA ?"
    LOA signifie **Letter of Acceptance** (lettre d'acceptation). Il s'agit du document par lequel une institution financière confirme formellement qu'une demande de crédit est approuvée, éventuellement sous conditions. C'est une étape intermédiaire entre la demande de crédit et la signature effective de l'acte notarié.

    Voir aussi le [glossaire](../concepts/glossary.md).
