# Dossiers de crédit — Vue d'ensemble

Les dossiers de crédit constituent le cœur de CreditSoft. Un dossier contient toutes les informations relatives à une demande de crédit : l'emprunteur, le montant demandé, le bien immobilier, les intermédiaires impliqués, les documents, les contrats et le suivi après acte.

## Qu'est-ce qu'un dossier de crédit

Un dossier passe typiquement par les phases suivantes :

1. **Création** — constitution du dossier initial avec l'emprunteur et le bien immobilier.
2. **En finalisation** — collecte des documents, complétion des données.
3. **À soumettre** — prêt à être envoyé à l'institution financière.
4. **Décision** — attente de l'approbation de l'institution financière.
5. **Acte** — acte notarié chez le notaire.
6. **Suivi après acte** — paiements, commissions, tâches post-acte.

Voir aussi : [Workflow et statuts](workflow-statuses.md).
