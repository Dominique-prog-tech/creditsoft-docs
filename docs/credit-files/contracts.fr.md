# Vue d'ensemble des contrats

Cet aperçu affiche tous les contrats de crédit — les suites des dossiers approuvés — sur toute leur durée.

## Ouvrir l'écran

Cliquez dans la sidebar sous **Crédits** sur **Vue d'ensemble des contrats**.
