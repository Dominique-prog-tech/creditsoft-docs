# Estimations

L'aperçu des estimations affiche toutes les estimations immobilières en cours ou archivées au sein de votre cabinet.

## Ouvrir l'écran

Cliquez dans la sidebar sous **Crédits** sur **Estimations**.

## Demander une estimation

!!! info "TODO"
    Procédure :

    1. Ouvrez le [bien immobilier](../crm/properties.md) ou le [dossier](../credit-files/edit-file.md).
    2. Cliquez sur **Demander une estimation**.
    3. Sélectionnez un [estimateur](../crm/appraisers.md).
    4. Envoyez la demande.

## Ajouter un rapport d'estimation

Une fois que l'estimateur vous remet le rapport, ajoutez-le au dossier :

1. Ouvrez l'estimation.
2. Cliquez sur **Charger le rapport**.
3. Sélectionnez le fichier PDF.
4. Saisissez la valeur estimée (tirée du rapport).
5. Cliquez sur **Enregistrer**.
