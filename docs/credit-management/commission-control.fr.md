# Contrôle des commissions

L'écran de contrôle des commissions vous aide à vérifier systématiquement si toutes les commissions attendues ont bien été reçues et correctement payées.

## Ouvrir l'écran

Cliquez dans la sidebar sous **Crédits** sur **Contrôle commissions**.
