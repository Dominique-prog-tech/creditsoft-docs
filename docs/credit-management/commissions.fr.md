# Gestion des commissions

L'écran de gestion des commissions vous permet de suivre toutes les commissions : à recevoir, reçues, à payer, payées.

## Ouvrir l'écran

Cliquez dans la sidebar sous **Crédits** sur **Gestion des commissions**.
