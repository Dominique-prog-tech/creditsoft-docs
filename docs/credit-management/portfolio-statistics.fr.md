# Statistiques du portefeuille

L'écran des statistiques offre une analyse approfondie de l'ensemble de votre portefeuille de crédits.

## Ouvrir l'écran

Cliquez dans la sidebar sous **Crédits** sur **Statistiques du portefeuille**.
